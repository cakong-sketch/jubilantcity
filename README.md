## How to Use

### Local Testing
1. Extract the ZIP file to your desired location
2. Open `index.html` in your web browser
3. The website will load with all images and styling

### Publishing to Web Server
1. Upload all files to your web hosting server
2. Ensure all files are in the same directory
3. Set `index.html` as the default/index page
4. Access your website via your domain name

### Important Notes
- Keep all files in the same directory for proper image loading
- The contact form uses the default email client (mailto)
- All images are included in the ZIP file
- No external dependencies or CDN links required

## Customization

### Changing Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #c41e3a;      /* Main red color */
    --secondary-color: #f0f0f0;    /* Light gray */
    --dark-color: #1a1a1a;         /* Dark color */
    --accent-color: #ff6b6b;       /* Accent red */
}
```

### Updating Content
- Edit text content directly in `index.html`
- Replace product images by uploading new files with the same names
- Update company information in the header and contact sections
- Modify product descriptions and features

### Adding More Products
- Copy the `.product-card` structure in the products grid
- Add new product images to the directory
- Update the image src attributes
- Maintain consistent styling by using existing CSS classes
